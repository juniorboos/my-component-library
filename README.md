## Built with

- React (Typescript)
- Storybook
- Rollup
- Github Actions
- Github Packages

# 🔥 Setup

- Clone the repo:

```
git clone https://github.com/juniorboos/my-component-library.git
```

## 🖥️ Frontend

- Install all dependencies and run:

```bash
$ npm i
$ npm run storybook # To run storybook
$ npm run build # To build the package
```
